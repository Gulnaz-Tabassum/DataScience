##How to Execute L2DCCA.py file:-

1.Downlod the .py(L2DCCA.py) file.

2.After downloading, install python 3(if alredy install than don't need to download the python 3).

3.Run the L2DCCA.py file.(or if you want to execute this file(.py) outside than call the MainFuncation():)

4.After executing L2DCCA.py open the dialog box.

5.dialog box ask for no of ROW for X and Y than put the no of rows of your matrix.

6.and put the matrix.

##what will do this L2DCCA.py file.

1.Transform of matrix of A(x) ,B(x) and A^x:={A^x+(A^x)^T}/2.

2.Calculate Ix^T{(sigma(i=1,N) sigma(j=1,N)A^x(lx^TXrx-lx^TXrx)Ay(ly^TYrx-ly^TXry)}ly.

3.find the agr max (lx^T Xr L^xy Yr^T ly).

4.Calculate [[0,XrLxyYr^t],[YrLxyXr^t,0]] and [[lx],[ly]].
