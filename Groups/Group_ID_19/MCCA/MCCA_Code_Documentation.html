MultiView Canonical Correlation Analysis (MCCA)

class MCCA(n_components=2,reg=0.001) [Source]

MCCA is a powerful technique for analyzing linear relationships between more (than two) sets of random variables.

It is an extension of CCA from two view scenario to include three or more number of data channels.

Parameters:

n_components: int, (default=2)

number of components to keep.

reg_param: float, (default=0.001)

a non-negative regularization coefficient added in case of high dimensionality features.

Attributes:

m -> number of views

n -> number of samples in each view

dimen: array,[m]

array of feature dimention of all views

C: array,[m,m]

Covariance matrix of each pair of views

weights: array,[m,n_components]

array of canonical covariate or projection vectors of each view

weights_tranform: array,[n,n_components]

array that stores the transformed views by applying the fitted model on it.

References:

1. The Multiview Canonical Correlation analysis model is made from: A Survey on Canonical Correlation Analysis Xinghao Yang, Weifeng Liu, Senior Member, IEEE, Wei Liu, Member, IEEE, Dacheng Tao, Fellow, IEEE. and

Methods:

1.fit(X,label)

fit model to data

Parameter:

X_list: 2darray, [n_sample,n_features][m]

array of m arrays of shape(n_sample,n_features)

2.transform(X_list)

apply the learned model to the data and transform it into required dimensionality form

Parameter:2darray, [n_sample,n_features][m]

X_list:

array of m arrays of shape(n_sample,n_features)

Returns:

X_reduced : 2darray, [n_sample,n_components][m]

which is the reduced dimensionality sample .

Examples:

MCCA import MCCA
                        import numpy as np

X1= np.array([[-1, -4,-5], [-25, -1,29], [-33, -24,-1], [1, 1,8], [23, 1,10]])
                        X2 = np.array([[-1, -1,2,3], [-2,2, -1,4], [-3,4, -2,5], [1,2, 1,7], [2,3, 1,7]])
                        X3= np.array([[-1, -1,3,2,5], [2, 3,-1,5,1], [-3, 4,2,7,4], [1, 1,3,1,3], [2, 1,3,4,-1]]) 
                         
                        X_list=np.array([X1,X2,X3])
                        mcca = MCCA()
                        mcca.fit(X_list)
                        X_reduced = mcca.transform(X_list)
